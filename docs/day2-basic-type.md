# Day 2. 기본형과 참조형

## 대원칙
> **자바는 항상 변수의 값을 복사해서 대입한다.**
> → 기본형은 그 값을 그대로 **!복사해서!** 전달하고, 참조형은 참조값을 **!복사해서!** 전달한다.

근데, 값은 말그대로 복사되면 2개가 되는 것이지만, 참조값은 복사되어서 2개가 되는 것이 아니라 하나의 주소로 이동하는 방법이 2가지가 된 것이다.

### 기본형(Primitive Type)

: int, long, doble, boolean 같은 기본 타입을 말함. 기본형은 어떤 값을 직접 대입하고 그 값을 바로 사용할 수 있음.

기본형은 자바가 기본으로 제공하는 데이터 타입이기 때문에 개발자가 새로 정의할 수 없음.

### 참조형(Reference Type)

: 값을 바로 저장하는 것이 아니라, 참조값을 저장함. 즉, 메모리 주소와 같은 것을 참조함. 참조형은 값을 바로 저장하는 것이 아니기 때문에 바로 사용할 수 없음. 그 참조값 안의 객체에 접근을 해야만 사용을 할 수 있다.

참조형인 클래스는 개발자가 직접 새로 정의할 수 있음.

- 클래스나 배열이 참조형 (클래스는 모두 참조형)
- 객체는 `.(dot)` 을 통해 메모리 상에 생성된 객체를 찾아가야 접근할 수 있음.
- 배열은 `[]` 를 통해 메모리 상에 생성된 배열을 찾아가야 접근할 수 있음.

→ 결국 메서드에서도 똑같은 것.

기본형: 메서드로 기본형 데이터를 전달하면, 해당 값이 복사되어 전달됨. 이 경우. 메서드 내부에서 매개변수의 값을 변경해도, 호출자의 변수 값에는 영향이 없다.

참조형: 메서드로 참조형 데이터를 전달하면, 참조값이 복사되어 전달됨. 이 경우, 메서드 내부에서 매개변수로 전달된 객체의 멤버 변수를 변경하면, 호출자의 객체도 변경된다.

### 변수

- 멤버 변수(필드): 클래스에 선언

    ```java
    public class Student {
    	String name;
    	int age;
    	int grade;
    }
    ```

  → `name` , `age` , `grade` 는 멤버 변수

- 지역 변수: 메서드에 선언, 매개변수도 지역 변수의 한 종류이다. / 특정 지역에서만 사용되는 변수

    ```java
    public class ClassStart3 {
    	 public static void main(String[] args) {
    	 Student student1; //변수 선언; Student 타입의 변수 student1
    	 student1 = new Student(); //생성; 생성된 실제 객체 주소를 student1 변수에 저장
    	 Student student2 = new Student(); //변수 선언과 생성을 동시에
    	 }
    }
    ```

  → `student1` , `student2` 는 지역 변수이다.

  *`new` 키워드는 **메모리에 실제 객체를 만들라**는 명령어


### 변수의 초기화

- 멤버 변수: 자동 초기화
    - 생성할 때 자동으로 초기화된다.
    - 숫자(int) = 0, boolean = false, 참조형 = null(참조할 대상이 없다는 뜻)로 자동 초기화된다.
    - 개발자가 초기값을 직접 지정할 수 있다.
- 지역 변수: 수동 초기화
    - 지역 변수는 항상 직접 초기화해야 한다.

### NULL

- null : 보낼 주소가 없다!

  *GC(Garbage Collection) - JVM의 GC는 아무도 참조하지 않는(NULL) 인스턴스를 자동으로 메모리에서 제거한다.

- `NullPointerException` 예외: null을 가리킬때 발생하는 예외
    - `.(dot)` 으로 참조값을 사용해서 객체를 찾아가는데, 참조값이 null일 때 이 예외가 발생한다.
    - → 쉽게 말해 `null`에 `.(dot)` 찍으면 `NullPointerException` 예외 발생

**정리:** `NullPointerException` 이 발생하면 `null` 값에 `.(dot)` 찍었다고 생각하면 문제 쉽게 찾을 수 있음
