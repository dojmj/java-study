# ☕️ Java Study – 김영한 자바 기본편 학습 기록

김영한 님의 **“자바 기본편”** 강의를 들으며  
코드를 직접 작성하고, 이해한 내용을 문서로 정리하는 저장소입니다.

📌 **목표:**  
- 자바 기본기 완전 정복  
- 개념 + 코드 두 축을 동시에 탄탄하게 만들기  
- 꾸준한 커밋으로 성장 기록 남기기  
- 나중에 면접/실무에서 바로 사용할 수 있는 참고 자료 만들기  

---

## 📚 Study Structure

이 저장소는 **두 가지 축**으로 구성되어 있습니다.

### 1) `src/` — 실습 코드
강의를 들으며 작성한 예제 코드, 실습 문제, 연습 파일 등을 보관합니다.

### 2) `docs/` — 개념 정리 (Markdown)

학습하면서 이해한 내용을 직접 정리합니다.

## 💡 Commit 메시지 규칙

가독성을 위해 날짜/섹션/내용 한눈에 보이도록 작성합니다.

예시:

docs - Day1: 클래스/인스턴스/향상된 for문 정리

src - Day1: 클래스와 인스턴스
